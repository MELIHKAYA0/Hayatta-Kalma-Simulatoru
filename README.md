# Karakter Tabanlı Hayatta Kalma Simülatörü

Bu proje, C programlama dili kullanılarak geliştirilmiş metin tabanlı (text-based) bir hayatta kalma oyunudur. Algoritma ve Programlama dersi kapsamında, temel programlama yapılarını (döngüler, koşul ifadeleri, operatörler) pratikte uygulamak amacıyla hazırlanmıştır.

## Özellikler

Proje içerisinde aşağıdaki teknik yapılar kullanılmıştır:
- **Do-While Döngüsü:** Oyunun ana döngüsü ve menü sistemi.
- **Switch-Case:** Kullanıcı komutlarının (A, S, R, E, F, P, X) yönetimi.
- **For Döngüsü:** Tehlike dalgası simülasyonu.
- **Mantıksal Operatörler:** Avlanma ve sığınak bulma şans faktörleri.

## Nasıl Oynanır?

Program çalıştırıldığında aşağıdaki komutlar kullanılabilir:
* **A:** Avlan (Enerji harcar, yemek bulma şansı vardır)
* **S:** Sığınak Ara (Enerji harcar, güvenli alan bulma şansı vardır)
* **R:** Dinlen (Sağlık ve Enerjiyi yeniler)
* **F:** Tehlike Simülasyonu (Hasar testi)
* **P:** Engel Aşma (Şifreli kapı simülasyonu)
* **E:** Envanter Görüntüle
* **X:** Çıkış
